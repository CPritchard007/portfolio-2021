Milestone Projects

T7 Chicken Legacy

An assistant applications for players of Tekken 7. As a newer player, it is important to be able to have the list of commands on hand; and with this application, you can! This application is a rebuild of the original T7ChickenPro, but one that will keep up-to-date with the new data released per season, as well as a fancy spotlight mode that will allow the user to view the combo via a gif.

T7ChichenLegacy is built using React Native, and will use json and redux for easy data management.

Pokemon Card Game Companion App

This application is used as an assistant to the pokemon card game. As most new players have very few copies of each card, most players tend to use a card in multiple decks. Using this application, the user can store all their decks in one place.

This application is created for IOS and makes use of the pokemon-tcg-api to get all the pokemon cards.
