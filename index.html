Pokemon TCG Companion App

This application is used as an assistant to the pokemon card game. As most new players have very few copies of each card, most players tend to use a card in multiple decks. Using this application, the user can store all their decks in one place.

This application is created for IOS and makes use of the pokemon-tcg-api to get all the pokemon cards.

Maple Crossing

Maple Crossing is an application that grabs data from both sides of the Windsor border. The application assists cross-border workers to find data such as travel delays, exchange rates, and border wait times.

The application was created using Flutter, allowing the application to be developed for both apple and android operating systems. This application also uses a backend API that was developed using PHP and Laravel.
